### Glossário Técnico

* **Arquitetura Cliente-Servidor:** Modelo de arquitetura distribuída onde as tarefas são divididas entre fornecedores de recursos ou serviços (servidores) e requisitantes de serviços (clientes).
* **CSS (Cascading Style Sheets):** Linguagem de folhas de estilo utilizada para definir a apresentação (cores, fontes, layout) de documentos escritos em linguagens de marcação como o HTML.
* **HTML (HyperText Markup Language):** Linguagem de marcação padrão para a criação de páginas web, responsável pela estrutura e semântica do conteúdo.
* **HTTP (Hypertext Transfer Protocol):** Protocolo de comunicação utilizado para transferir dados (como textos HTML, imagens, vídeos) na World Wide Web. É a base da comunicação de dados na Web.
* **Internet:** A infraestrutura global de redes de computadores interconectadas que utilizam o conjunto de protocolos TCP/IP para servir bilhões de usuários. É o "caminho" por onde a Web trafega.
* **TCP/IP (Transmission Control Protocol/Internet Protocol):** Conjunto de protocolos de comunicação que implementam a pilha de protocolos sobre a qual a Internet e a maioria das redes comerciais funcionam.
* **URI (Uniform Resource Identifier):** Uma sequência de caracteres compacta usada para identificar ou denominar um recurso na Internet. A URL (Uniform Resource Locator) é um tipo específico de URI que também indica como localizar esse recurso (ex: `http://www.exemplo.com`).
* **World Wide Web (WWW):** Sistema de documentos em hipermídia que são interligados e executados na Internet.
